# Steel Equal Leg Angles - Coremark Metals

## Overview
A36 Hot Rolled Steel Equal Leg Angles for structural applications including bracing, supports, trailers, and brackets.

- **Grade:** ASTM A36
- **Type:** Hot Rolled Steel
- **Finish:** Mill Finish
- **Yield Strength:** 47,700 psi
- **Tensile Strength:** 58,000 psi
- **Properties:** Weldable, formable, machinable

## Usage Notes
1. Each size configuration should be saved as a separate .SLDLFP (SolidWorks Library Feature Part) file
2. File naming convention: `[Size]x[Size]x[Thickness].SLDLFP` (e.g., `1x1x0.125.SLDLFP`)
3. The sketch for each profile should be fully dimensioned with the angle cross-section centered at the origin
4. Material properties should be set to ASTM A36 Steel in the file properties

## Custom Properties Template
Each .SLDLFP file should include the following custom properties:
- **Material:** ASTM A36 Steel
- **Supplier:** Coremark Metals
- **Weight per Foot:** [value] lbs/ft
- **Price per 20ft:** $[value]
- **Leg Size:** [value] inches
- **Wall Thickness:** [value] inches
- **Description:** Hot Rolled Steel Equal Leg Angle
