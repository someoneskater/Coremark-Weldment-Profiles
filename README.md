# Coremark-Weldment-Profiles
SolidWorks weldment profile library with 558 profiles from Coremark Metals including Steel, Aluminum, and Stainless Steel structural shapes with integrated pricing data.

## Tools

### CreateProfiles.bas
A VBA macro for SolidWorks that provides automated functions to draw weldment profile sketches including:
- Angle profiles (L-shaped)
- Rectangular/Square tube profiles
- Round tube profiles (pipes)
- Channel profiles (C-shaped)

See [CreateProfiles_README.md](CreateProfiles_README.md) for detailed documentation and usage instructions.
