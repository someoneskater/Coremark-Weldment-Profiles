# Coremark-Weldment-Profiles
SolidWorks weldment profile library with 558 profiles from Coremark Metals including Steel, Aluminum, and Stainless Steel structural shapes with integrated pricing data.

## Profile Categories

### Aluminum
#### Angles
- **Material:** 6061-T6
- **Size Range:** 3/4" to 8"
- **Price Range:** $35 - $1,853
- **Available Sizes:** 3/4", 1", 1.5", 2", 2.5", 3", 3.5", 4", 5", 6", 7", 8"
- **Details:** [View Aluminum Angles Specifications](Aluminum/Angles/README.md)

## Usage
This repository contains documentation and specifications for Coremark Metals weldment profiles. Each profile category includes:
- Detailed material specifications
- Size and dimension information
- Pricing ranges
- Material properties
- Application guidelines

For detailed specifications, navigate to the specific profile directory.
