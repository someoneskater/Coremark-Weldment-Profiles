# Coremark Metals Weldment Profile Catalog

## Overview

This catalog documents all 613 structural profiles included in the Coremark Metals SolidWorks weldment library, organized by material type and shape.

## Materials Covered

- **Steel** - Structural carbon steel (ASTM A36, A992)
- **Aluminum** - 6061-T6 and 6063-T5 structural aluminum
- **Stainless Steel** - 304/304L and 316/316L stainless steel

## Profile Types

### 1. Angles (Equal and Unequal Leg)
L-shaped structural sections used for bracing, framing, and support applications.

### 2. Channels
C-shaped or U-shaped sections used for framing, bracing, and structural support.

### 3. Beams
I-shaped and W-shaped sections providing excellent bending strength for load-bearing applications.

### 4. Square Tubes
Hollow square sections offering uniform strength in all directions.

### 5. Rectangular Tubes
Hollow rectangular sections providing directional strength characteristics.

### 6. Round Tubes
Hollow circular sections ideal for aesthetic applications and torsional loads.

### 7. Pipes
Schedule-rated circular sections designed for both structural and fluid-carrying applications.

### 8. Tees
T-shaped sections used as stiffeners and structural components.

## Dimensions and Nomenclature

### Angle Notation
Format: `LegSize x LegSize x WallThickness`
- Example: `2x2x0.125` = 2" x 2" angle with 0.125" wall

### Channel Notation
Format: `C + Depth x Weight`
- Example: `C6x10.5` = 6" channel at 10.5 lbs/ft

### Beam Notation
Format: `W + Depth x Weight`
- Example: `W12x35` = 12" wide flange beam at 35 lbs/ft

### Tube Notation
Format: `Width x Height x WallThickness` or `OD x WallThickness`
- Square: `4x4x0.250` = 4" x 4" with 0.250" wall
- Rectangular: `4x2x0.188` = 4" x 2" with 0.188" wall
- Round: `2x0.120` = 2" OD with 0.120" wall

### Pipe Notation
Format: `NominalSize-Schedule`
- Example: `2-Sch40` = 2" nominal diameter, Schedule 40

### Tee Notation
Format: `WT + Depth x Weight` or `T + Size x Size x Wall`
- Steel: `WT5x11` = 5" tee at 11 lbs/ft
- Aluminum: `T3x3x0.1875` = 3" x 3" tee with 0.1875" wall

## Usage Notes

### Material Selection

**Steel** - Best for:
- High strength requirements
- Cost-effective solutions
- Heavy load applications
- General construction

**Aluminum** - Best for:
- Weight reduction critical
- Corrosion resistance (non-marine)
- Aesthetic applications
- Easy machinability

**Stainless Steel** - Best for:
- Corrosive environments
- Food processing equipment
- Chemical resistance required
- Sanitary applications
- Marine environments

### Standard Specifications

- **Steel:** ASTM A36 (general), ASTM A992 (wide flange beams)
- **Aluminum:** 6061-T6 (structural), 6063-T5 (architectural)
- **Stainless:** 304/304L (general), 316/316L (enhanced corrosion resistance)

### Design Considerations

1. **Load capacity:** Always verify structural calculations
2. **Corrosion:** Select appropriate material for environment
3. **Welding:** Follow AWS standards for welding procedures
4. **Cost:** Balance performance requirements with budget
5. **Availability:** Confirm stock availability with Coremark Metals

## Profile File Format

All profiles are provided as `.sldlfp` (SolidWorks Library Feature Part) files containing:
- 2D cross-section sketch
- Material properties (density, strength)
- Standard dimensions
- Reference to industry standards

### File Naming Convention
Files are named by their size designation for easy identification:
- `2x2x0.125.sldlfp`
- `W12x35.sldlfp`
- `C6x10.5.sldlfp`
- `4-Sch40.sldlfp`
